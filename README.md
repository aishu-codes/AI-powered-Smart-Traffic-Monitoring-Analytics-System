## 📌 Project Overview

This project performs real-time traffic monitoring using AI-based vehicle detection and traffic analytics.

The system can:

- Detect vehicles from traffic video/CCTV
- Count vehicles in real time
- Analyze traffic density
- Recommend smart signal optimization
- Detect emergency vehicles
- Detect wrong-direction movement
- Identify possible accident situations
- Store analytics data into CSV
- Visualize insights using Tableau dashboards
